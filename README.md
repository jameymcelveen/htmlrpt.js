### From Source
-----------
1. Clone SSH Repo:
   [https://github.com/jameymcelveen/htmlrpt.js](https://github.com/jameymcelveen/htmlrpt.js)
2. Install Packages:
    ```
    cd htmlrpt.js
    npm install
   ```
3. Build:
    ```
    npm build
    ```
4. Copy:
   You can now copy `htmlrpt.js` or `htmlrpt.min.js` from the `./htmlrpt.js/lib` folder to your assets folder.
